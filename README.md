# 🚀 Quick Start

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/toolbox-pro.git
cd toolbox-pro
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
